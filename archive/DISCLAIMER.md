# Disclaimer

This repository is an **engineering portfolio** documenting design, implementation, and operations work on the **CXR** platform. It is intended for professional review (recruiters, engineering managers, clients, collaborators) and personal record-keeping.

## Not production

- Configurations described here target **local development** and **bootcamp lab** environments.
- **HashiCorp Vault dev tokens**, default Grafana passwords, and similar values are **lab-only** and must never be used in production.
- Kubernetes, Terraform, and some compose stacks are **scaffolds** unless explicitly marked as deployed.

## Data
